Install Theme

To start you must have a working version of WordPress. Almost all major web hosts provides one click WordPress install and you can consult your host's help site for more information in this regard. For manual install you can consult WordPress Codex Installing WordPress.

Once you have a working version of WordPress, You need to download "All Files and Documentation" from themeforest download page and extract the downloaded zip to get various files including inspiry-startable.zip.

Install Theme Via WordPress Dashboard ( Recommended Way )

  1. Go to 'Appearance > Themes' section
  2. Click 'Add New' and select the 'Upload Theme' option
  3. Click 'Upload Theme'
  4. Choose the inspiry-startable.zip file and press 'Install Now'
  5. Once the theme is uploaded you need to activate it.
  6. After activation click on the 'Being activating plugin' button to install Inspiry Startable Framework plugin (required).
  7. Check the Inspiry Startable Framework plugin box, select 'Active' action and apply.
  8. After activation go to "Appearance > Customize" to play with theme's settings.

if you face any problem during upload through dashboard, please upload the theme using FTP as guided below.

Install Theme Via FTP ( Alternative Way )

  1. Access your hosting server using an ftp client like FileZilla
  2. Go to the 'wp-content/themes' folder of your WordPress installation
  3. Extract the inspiry-startable.zip file and put the inspiry-startable folder in 'wp-content/themes/' folder
  4. Go to 'WordPress Dashboard > Appearance > Themes' section to activate the theme
  5. After activation go to "Appearance > Customize" to play with theme's settings.

Install Child Theme ( Highly Recommended )

It is highly recommended to use child theme, So that you can easily update parent theme whenever a new update becomes available. A pre-built ready to use child theme is included in the main theme pack that you have downloaded from themeforest.

To install child theme, You need to upload the inspiry-startable-child.zip and activate it in the same way as you have uploaded and activated parent theme's inspiry-startable.zip.

Add News Post

Go to WordPress Admin > Posts > Add New
  1. Provide the post title
  2. Provide the post contents
  3. Add or select the related category
  4. Add or select the related tags
  5. Provide the featured image. The image should have minimum width of 750px and minimum height of 393px. The bigger size images will be cropped automatically.
  6. Publish the post once it is ready.

Add Portfolio Post

Go to WordPress Admin > Portfolio > Add New
  1. Provide the portfolio title
  2. Provide the portfolio contents
  3. Add or select the portfolio item types
  4. Add portfolio item details
  5. Provide the portfolio item featured image. The image should have minimum width of 1100px and minimum height of 480px. The bigger size images will be cropped automatically.
  6. Publish the portfolio item post once it is ready.

Add Service Post

Go to WordPress Admin > Services > Add New
  1. Provide the service title
  2. Provide the service contents
  3. Add or select the service types
  4. Add service excerpt that will be shown on the homepage and services listing page for this service post.
  5. Provide the service icon sized featured image. The bigger size images will not be cropped.
  6. Publish the service post once it is ready.

Add Team Member Post

Go to WordPress Admin > Team > Add New
  1. Provide the team member title
  2. Provide the team member short description
  3. Add team member details.
  4. Provide the team member featured image. The image should have minimum width of 400px and minimum height of 410px. The bigger size images will be cropped automatically.
  5. Publish the team member post once it is ready.

Add Testimonial Post

Go to WordPress Admin > Testimonials > Add New
  1. Provide the testimonial author name
  2. Provide the testimonial details
  3. Publish the testimonial post once it is ready.

How to set up FAQs Page

I have developed Quick and Easy FAQs plugin for this purpose.

  1. Install the Quick and Easy FAQs plugin. As guided in screenshot below.
  2. After installing and activating the plugin. You can add FAQs as displayed in screenshot below.
  3. Later on you can use following shortcodes to display FAQs on a page.
    • Display all FAQs in simple list style. Shortcode [faqs]
    • Display all FAQs in simple list style but separated by groups. Shortcode [faqs grouped="yes"]
    • Display FAQs in simple list style but filtered by given group slug. Shortcodes [faqs filter="group-slug"] or [faqs filter="group-slug,another-group-slug"]
    • Display all FAQs in toggle style using following shortcode. Shortcode [faqs style="toggle"]
    • Display all FAQs in toggle style but separated by groups. Shortcode [faqs style="toggle" grouped="yes"]
    • Display FAQs in toggle style but filtered by given group slug. Shortcode [faqs style="toggle" filter="group-slug"] or [faqs style="toggle" filter="group-slug,another-group-slug"]
    • Display all FAQs in filterable toggle style. Shortcode [faqs style="filterable-toggle"]

How to Modify Styles

There are multiple ways you can modify the theme styles.

  1. Quick CSS - Go to Appearance > Customize and navigate to Styles to add any quick css.
  2. Custom CSS in Child Theme - 'child-custom.css' file is provided in child theme to add major css changes.

    Note: It is highly recommended to use child theme for any changes that you want to make as this way you can update parent theme whenever an update becomes available. Simply copy a template file that you want to change to child theme after creating same directory structure and then modify it. For more info http://codex.wordpress.org/Child_Themes.

  3. Custom CSS in Parent Theme - if you are not using child theme then 'css/custom.css' file is provided in parent theme to add major css changes.
